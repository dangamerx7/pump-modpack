## 📥 Installation 

Download the modpack from <a href="https://modrinth.com/modpack/pump-modpack/version/0.0.1">here</a> and then use launchers like Prism Launcher or ATLauncher to extract it into a launchable format. I recommend using Prism Launcher, simply drag the modpack into Prism Launcher and install it.

## 🔍 Features

- Better Performance: Sodium-fabric, Lithium-fabric, ImmediatelyFast, Dynamic FPS, Methane, and Krypton can help improve game performance on low-end PCs by optimizing your game.

- Enhanced PvP Experience: BadPackets-fabric, VanillaIcecreamFix, TotemCounter, and Client Sided Crystals can enhance the PvP experience by fixing game bugs, packet problems, and improving crystal PvP.

- Quality of Life Improvements: ChatHeads, Appleskin-fabric, BetterBeds, BetterHurtCam, MoreCulling, and NotEnoughCrashes enhance the overall gameplay experience by adding new features like heads-up displays, better bed mechanics, and improved crash reporting.

- Building Assistance: Litematica-fabric can help players create better buildings and structures by allowing them to save and load building templates, as well as displaying holographic overlays for easy block placement.
